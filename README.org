* Description
An extended Hugo blog theme based on [[https://github.com/lxndrblz/anatole][anatole]]. This theme brings a clean, minimalist design with additional features and customization options for your blog.

Check out a [[https://lisp.netlify.app][live demo]] to see it in action.

* Installation
Add this theme as a submodule in your Hugo project:

#+begin_src shell
mkdir themes
git submodule add https://github.com/conao3/anatole-ext.git themes/anatole-ext
#+end_src

Then update your =config.toml= to use the theme:

#+begin_src toml
theme = "anatole-ext"
#+end_src
